## 1. Purpose & Strategy

### Target Audience
React, Svelte, Vue 프레임워크를 사용하는 프론트엔드 개발자. 레트로/노스탤지어 UI를 구현하고 싶거나, 독특한 디자인 시스템을 찾는 개발자.

### Problem
Windows 7 스타일 UI를 직접 구현하려면 CSS와 컴포넌트 구조를 처음부터 만들어야 하고, 프레임워크별로 따로 개발해야 함.

### Solution
CLI로 필요한 컴포넌트만 설치하여 프로젝트에 바로 사용. shadcn/ui 스타일로 소스코드가 프로젝트에 복사되어 커스터마이징 자유도 높음.

### Value Proposition
"Windows 7 UI를 React, Svelte, Vue에서 한 줄 명령으로"
